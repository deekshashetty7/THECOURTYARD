
  # Brain Training App

  This is a code bundle for Brain Training App. The original project is available at https://www.figma.com/design/eKVCFG62bOXHPMZqhrNDmY/Brain-Training-App.

  ## Running the code

  Run `npm i` to install the dependencies.

  Run `npm run dev` to start the frontend and backend together.

  Run `npm run dev:client` for frontend only, or `npm run dev:server` for backend only.
